Container management

To display the list of containers:

1.Click Administration > Processes > Containers.

2.Specify the search parameters and click Search.

The search results are displayed. The additional information about the containers are displayed the details pane tabs.

Tab

Description

Details

General information about the selected container

Processes

List of processes included in the container. There are buttons to specify the processes’ parameters

Parameters

List of parameters gathered from all processes included in the container

Files

List of files of all processes included in the container. There are buttons to specify file’s parameters.

Schedule

Schedules created for the container. You can set several active schedules for a container. There are buttons for schedule management

Roles

Roles that allow a user to run the container manually
