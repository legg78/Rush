Communication standard parameters and profiles

Each communication standard requires a set of parameters that identify a particular standard and defines the processing conditions for entities that use that particular standard in the network. The values for these parameters must be specified at the communication device level. Optionally, values may also be defined at the host or terminal level, in which case these values override those parameters defined at the communication device level.

Note: Default values may be set at the level of the standard, which are used if values are not defined for a particular user of the standard.

 

Profiles facilitate configuration cases where multiple entities exist, making it cumbersome to configure each standard individually. A profile, which is assigned to an object (terminal), contains and defines its settings relating to the communication standard.
