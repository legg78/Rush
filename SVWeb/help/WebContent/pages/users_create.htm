Registering a new user

User can be registered manually or using an application (see User management using an application). To register a user manually:

1.Click Administration > Permissions > Users.

2.On the search result pane, click Add.

3.Specify the user parameters.

Parameter

Description

Login

The user’s account used to log in to the system.

Password

The password used to log in by the user.

Person

The person to be associated with the user. To set a person, click and select the required person from the registered ones (the Persons page). Choose the person and click Select.

If the required person cannot be found, create a new one by clicking New and setting the person data.

Role

The user’s role that corresponds to his or her job responsibilities. You must select one role. If the user has several roles then you should specify them later.

Default institution

The institution to which the new user belongs.

4.Click Save.

The user is registered with Active status.

5.Specify additional institutions/agents where the user will act.

6.Specify additional roles in accordance with the user’s job responsibilities.
