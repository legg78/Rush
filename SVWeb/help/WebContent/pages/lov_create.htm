Creating lists of values

A list of values is used to store a set of values that a specific parameter can possess. This enables a user to select the required value from the list of predefined values in order to configure the parameter.

1.Click Configuration > Arrays > Lists of values.

2.Specify search parameters and click Search.

The search result pane displays the lists that correspond to the set parameters. There are control buttons on the bottom of the pane to maintain lists. The details pane displays general data of the selected lest of values.

3.To create a new list of values (LOV), click Add on the search result pane.

4.Specify the list parameters:

Parameter

Description

Language

The language of the input text data.

Name

The list name.

Module code

The code of the module that will use the list.

Data type

The type of data for the lists’ elements.

Dictionary

Specifies whether the list is paged from a dictionary or select the LOV query – if elements of the list are defined from the SQL query to the database.

Sort mode

The rule of elements ordering (for example, by code, by name).

Appearance

The display mode for the lists’ elements (for example, code+name, name+code, code, name).

Parameterized

Specifies whether the list will have a set of parameters.

5.Click Save.
