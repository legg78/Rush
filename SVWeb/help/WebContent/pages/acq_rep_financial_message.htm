Financial messages recommended for payment

Report name — Financial messages recommended for payment.

Report description — Generated on each payment system (China UnionPay, MasterCard, Visa, Diners) for a specific SVBO clearing date. Note that a report for VISA payment system contains only DMS transactions.

This report is intended to be used during process administration; it consists of two parts:

1.The financial messages recommended for payment on an SVBO clearing date (with regards to payment systems and acquiring institutions).

2.A report on Reject operations for payment systems and acquiring institutions on a specific SVBO clearing date.

Part one

Report header 1. The financial messages for submitting a payment on the SVBO clearing date of a specific payment system name.

Used to generate postings on CASH/SALE acquiring operations and for verifying compensation from payment systems. The report is generated on the results of SVBO clearing and after receiving a payment notification from the relevant payment systems (Union, MasterCard, Diners).

Part two

Report header 2. Repeatedly submitted and paid reject for date SVBO clearing date on payment system payment system name.

Used for accounting Reject operations and verification with compensation from payment systems. Generated on the results of notification about payment for provided operations from payment systems (Union, MasterCard, Diners).

Building parameters and field content is the same for both parts of the report.
