Auditing user actions

SmartVista monitors and records all user activities in audit logs. These logs may be viewed by users with the necessary privileges.

1.Click the Monitoring > Audit logs.

 

2.Specify the search parameters in the search pane and click Search.

The matching audit log records are displayed in the search result pane. The following information is provided.

Parameter

Description

Privilege

The user’s privilege.

Action date

Date and time of the user’s action. Time is displayed accurate to one microsecond.

Entity type

The system object type.

Object ID

System object identifier.

User

The user’s login.

Status

The action status (successful of failed).

The Details tab provides detailed information about a record selected in the list:

Parameter

Description

Column name

The name of the modified column

Data type

Type and format of modified data.

Old value

The previous parameter value (for updating and deleting)

New value

The new parameter value (for updating and inserting)

Specific actions are performed based on the object:

Object

Action

Added object

A record with the INSERT type is created.

Deleted object

A record with the DELETE type is created.

Modified object

A record with the UPDATE type is created.

Data on user’s sessions is displayed on the Info tab:

Parameter

Description

User name

The name of the user who performed the action

Start time

Date and time when the action was started.

End time

Date and time when the action was finished

IP address

The address of the user’s PC in a computer network used to access the system
