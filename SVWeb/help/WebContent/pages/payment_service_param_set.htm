Specifying service parameters

1.Click Payment orders > Service providers.

2.Specify search parameters and click Search.

3.Select the target service provider from the search results.

4.Go to the Services tab on the details pane.

5.To add a payment parameter, click Add on the right part of the tab.

6.In the New parameter window, specify the payment parameter settings.

Parameters

Description

Parameter

Select the previously registered parameter name.

Data type

Specify the parameter’s data type.

LOV

Specify the code identifying a list of values (LOV) whose values can be used as the parameter value.

Order stage

Stage when the parameter should be transferred.

Mandatory check

Select if payment cannot be performed with the unset parameter.

Fixed

Select to put the parameter to the template.

Editable

Select to enable the parameter to be editable.

Default value

Specify the default value for the parameter.

Display order

Specify the order in which the parameter is displayed in the list. This is only used for sorting.

5.Click Save.
