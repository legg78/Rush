Setting the fee value

Fee values are set at the level that was specified as the definition level of the corresponding service term: service, product and object, or object.

To set the fee value:

1.Access the page that corresponds to the service term definition level.

The table below shows the correspondence of service term definition level with the page on which the setting is performed.

2.Click Search on the search pane.

3.Select the service/product/object (for example, Card, Account, Terminal) to set the service term value.

4.Click the Service terms details pane tab.

A warning icon means that the service term cannot be set at this page (level):
 

5.Select the service term of the Fee type and click Set (see on the right side of the tab).

6.Specify the fee parameters value:

Parameter

Values

Create new value

Set all parameters to new values.

Clone value

Duplicate the previously set value. Thresholds and period of validity can be edited (that is, all parameters except the Currency, Fee rate calculation and Fee base calculation)

Apply value

Duplicate the previously set value. Only the period of validity (Start date and End date) can be edited

Fee

From the list, select the fee whose value you wish to duplicate. The box is available if the Clone value or Apply value radio button were selected. After the fee has been selected, the parameter values are set in the following fields.

Currency

From the list select the currency in which the commission is charged. On setting fee calculation, the user specifies the result currency. If this currency differs from the fee currency, the fee is converted into the desired currency. A currency rate type should be specified for each fee type. This rate type is used to convert the fee amount if its and the required currencies are differ

Note: For fees with limits, the same currency must be used for both the fee and the limit.

Fee rate calculation

Select the method of the fee rate definition.

Fee base calculation

Select the method of the fee base definition. The list is disabled for fixed rate fees.

Note: Depending upon the selected Fee rate calculation and Fee base calculation, parameters the fee tier parameters will be specified.

7.In the table, specify the fee tiers. Click Add and specify the new fee tier parameters.

Parameter

Description

Count threshold

Minimum of transactions for applying the fee. The maximum value is the Count threshold parameter value of the next tier.

Sum threshold

Minimum sum of operation(s) for applying the fee. The maximum value is the Sum threshold parameter value of the next tier.

Fixed rate

Fixed rate of the fee.

Min value

Minimum value of the fee.

Max value

Maximum value of the fee.

Length type

Fee charge periodicity, rate period (for fee by a period)

8.To set fee tiers, you can use the Fee tier wizard (for more details, see Using the fee tier wizard).

9.In the Start date and End date fields, specify the fee validity. If the end date is not set then the fee is an indefinite validity period.

If the scale was set for the service term, then the Modifier list is displayed below the period of validity fields. The user can select the previously created condition on which the service term value is applied.

For a description of the Limit and Cycle tabs, see limits and cycles.

8.Click Save.
