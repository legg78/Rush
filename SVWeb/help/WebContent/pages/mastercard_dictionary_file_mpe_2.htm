MasterCard dictionary file (MPE) loading

Process ID and name

10000013 - MasterCard dictionary file (MPE - Merchant Payments Ecosystem) loading

Process function

The process loads MasterCard MPE (Merchant Payments Ecosystem) dictionary file. Data of this file is used to process messages from MasterCard.

Process parameters

Parameter

Description

Mandatory/Optional

Network ID (I_NETWORK_ID)

A payment network identifier.

O

Institution (I_INSTITUTION_ID)

A destination financial institution identifier. If a definite institution need not be specified, select 9999 = Default/All.

O

Card owner network identifier (I_CARD_NETWORK_ID)

An identifier of a card owner network.

M

Card owner institution (I_CARD_INST_ID)

A card owner financial institution identifier. If a definite institution need not be specified, select 9999 = Default/All.

O

MasterCard MPE table (I_TABLE)

A name of the MPE table to be loaded. The default value is IP00FULL - Full.

O

Expansion (I_EXPANSION)

Expanded Member Info.

O

Files

The process uses incoming MPE files in text format.
