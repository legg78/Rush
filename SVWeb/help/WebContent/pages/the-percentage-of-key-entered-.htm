The percentage of key-entered (high-risk)

Report name – The percentage of key-entered (high-risk) Authorization as a percentage of all Authorizations, within a defined period exceeds a predefined limit.

Report description – The percentage of key-entered (high-risk) Authorization counts to the total volume of all Authorization counts in a certain time period (example period = 1 day, 1 month).

Report header – The percentage of key-entered (high-risk) Authorization as a percentage of all Authorizations, within a defined period exceeds a predefined limit.

Report building parameters
