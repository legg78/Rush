Configuring user contacts

During the registration of a user, a system administrator inputs the user’s contact and personal data. Users can view their own data, and add, modify or delete contact details.

1.Click User settings on the pane.

2.On the Settings page, select the Contacts tab. The tab is divided in two: the left displays the type of contact (for example, Director), and the right displays the mode of communication (for example, email address, telephone number, instant message). Data can be sorted by contact type.

3.To add a new contact, click the Add button labeled 1 in the screen capture.

4.In the New contact box, specify the contact details.

Parameter

Description

Preferred language

User interface language.

Contact type

Type of a contact to be dedicated for the user. Contact types are selected from the contact type (CNTT) dictionary configured in the system.

Person

A real person associated with the contact. Selected from the list of real persons registered in the system.

Position

Organization position (job role) occupied by the user.

Communication type

Mode of communication to be used to contact the user. The value is selected from the CMNM dictionarywhich contains such means of communications as mobile phone, email, fax, skype, messengers and other.

Communication address

Address to be used for communication with the user.

 

5.Click Save.

6.To add a communication type for a contact type, select the contact in the left pane and click the Add button labeled 2 in the screen capture.

7.Specify the communication type and address in the relevant fields.

Note: Personal information on the user currently logged on the system is displayed on the Personal data tab of the User settings page.
