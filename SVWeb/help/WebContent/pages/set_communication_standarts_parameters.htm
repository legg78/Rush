Setting communication standards

Communication standards specify the format of messages (or files) and the rules of their exchange. Communication standards are typically established by standards bodies, such as the International Organization for Standardization (ISO), or international payment systems. For more details about communication standards, see the Communication interfaces.

The Standards page allows you to configure the communication standard (or interface) that SmartVista uses to communicate with external entities. The configuration performed in SVBO is transferred to SmartVista Front End.

1.Click Administration > Communication > Standards.

2.Specify the search parameters and click Search.

The result search pane displays any communication standards matching the search parameters.

Parameter

Description

ID

Standard identifier and name.

Standard type

Standard type denoting its application:

•Network basic standard

•Network clearing standard

•Network communication standard

•Terminal communication standard

Online plugin

The code and the name of the online connection plugin. Online plugins enable parameters specific to particular online connections to be configured.

Response codes

Response codes are used to map response codes in external messages and files (as defined by standards such as ISO 8583) to SmartVista response codes.

Key types

Key types used to map standard key types (as defined by EFT protocols) to SmartVista cryptographic keys.

3.Configure the communication standard using the following buttons on the search result pane.

Button

Description

Add

Register a new communication standard.

Add version

Add a new version for the selected standard.

Edit

Modify values of the standard parameters.

Reorder

Set the sequence in which the standard versions are displayed.

Delete

Delete the selected standard.

The details pane tabs provide detailed information about the selected standard.

Tab

Description

Details

General description of the standard (protocol) and the specific version.

Parameters

Lists the parameters of the standard and, if applicable, specific versions of the standard.

Response codes mapping

Displays the mapping of response codes defined in the selected standard to the corresponding SmartVista response codes. This setting is applicable only to communication standards.

Key types mapping

Displays the mapping of key types defined in the selected standard to the corresponding SmartVista key types. This setting is applicable only to communication standards.

Objects

Lists the objects within the SmartVista system to which standard applies. Such objects may, for example, include network hosts, and communication devices.

Payment formatters

Payment formatters specify how merchant or service provider messages are formatted.

Operation type mapping

Displays the mapping of transaction types defined in the selected standard to the corresponding SmartVista transaction types. The network transaction type is determined by the relevant network standard specification (for example, Visa, MasterCard).

Message type mapping

Displays the mapping of message types defined in the selected standard to the corresponding SmartVista message types.
