Viewing debts

The Debts page displays information relating to the debts of a specific customer account within a specified time period.

1.Click the Issuing > Credit > Debts menu items. The Debts page will be displayed.

 

2.Specify the search parameters. To list all records, leave the filter fields blank.

Parameter name

Description

Institution

The Institution that is associated with the account

Card number

The card number (PAN)

Account number

The customer account number

Operation type

The operation type for the debt being viewed

Status

Active – the debt is taken into account on invoices, interests are charged on the debt, the debt can participate in the repayment.

Suspended – no interests are charged and the debt cannot be repaid. It can be transferred into the Active status while continuing to work with ordinary debt.

Paid – the debt is fully repaid and cannot be transferred into any another status.

In collection – the debt repayment has not yet been processed.

Canceled – the debt was not repaid, but was either reversed or disputed.

Date from

Start Date

Date to

End date

3.Click Search.

The search result panel will display the debts that correspond to the specified parameters.

The details panel tabs show general and additional information about the selected debt.

Tab

Description

Details

General information.

Interest

Information about the interest that was charged on the debt.

Payments

Information about payments for the debt.
