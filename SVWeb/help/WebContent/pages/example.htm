Example: configuring a rate with a rate pair

1.Create two rate types (RATE_TYPE_1 and RATE_TYPE_2), between which we must define the dependence.

oRATE_TYPE_1 – reversible, uses cross-exchange rate.

oRATE_TYPE_2 – non-reversible, uses direct exchange rate.

2.Specify the rate pair for the first rate.
The RATE_TYPE_1 rate is calculated as the following: <RATE_TYPE_2_value> * 12 / 10.

3.Specify the same corresponding dependence for the second rate.
The RATE_TYPE_2 rate is calculated as the following: <RATE_TYPE_1_value> * 10 / 12.

4.Set up the exchange rate (for converting USD to EURO) for the RATE_TYPE_1 rate.

The Rates added window displays information on the created exchange rates:

1 – rate of the RATE_TYPE_1 type for converting USD to EURO.

2 – rate of the RATE_TYPE_2 type for converting EURO to USD (was added automatically in accordance with the specified dependence).

3 – rate of the RATE_TYPE_1 type for converting USD to EURO (was added because the RATE_TYPE_1 rate type is reversible).
