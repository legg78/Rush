China UnionPay Disputes

The Disputes page displays data on dispute messages received from the China UnionPay system. China UnionPay dispute files are downloaded to the system by the CUP load dispute file process.

1. Select Payment System > China Union Pay > Disputes.

2.Specify the search parameters in the search pane:

Parameter

Description

Rrn

Research reference number.

PAN

Card number (PAN). You can use a wildcard (*) and type in a part of the number.

Transaction amount from

Transaction amount range - initial value

Transaction amount to

Transaction amount range - maximum value

Transaction date from

Transaction period - initial date

Transaction date to

Transaction period - final date

Session ID

The identifier of the file load session (a numeric value is entered manually).

The search result pane displays a list of messages according to the specified conditions.

Additional information about a selected message is available in the Details, Clearing operation and Authorization tabs under the list of messages.

The transaction settlement date in the YYYYMMDD format.

Identifies the name of the document file as described in ISO/IEC 7816-4. Indicated with 32 visible hexadecimal characters (0–9, A–F). If the length of the effective value is less than 34, the subsequent content should be blank.

Indicates whether the message is double or single. Specified by the sender in accordance with the original transaction:

A3-digit code indicating a transaction type from the dual-message settlement file or the IC card offline purchase file.
