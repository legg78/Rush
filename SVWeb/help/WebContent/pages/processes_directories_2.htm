Directories

A directory is a file system cataloging structure which contains references to other files, and possibly other directories.

To add a new file directory:

1.Click Administration > Processes > Directories.

2.Click Add.

3.In the New Directory window, configure the following fields:

Field name

Description

Name

Path name

Encryption type

Determines whether disk encryption is used. This parameter value must be specified to comply with PA-DSS requirements:

•DENCTRUE - Encrypted — use this option if disk encryption is used (the encryption is performed with external tools)

•DENCFALSE - Not encrypted — use this option if disk encryption is not used

Folder path

Folder path. The path can be configured using the $ROOT_DIR variable; for example: $ROOT_DIR/dir1/dir2/, where $ROOT_DIR is the Root folder parameter value specified in the Input/Output section of Administration > Settings > Settings (for example, /home/weblogic12). The system checks the existence of the directory; if the specified directory is not found, a warning message is displayed.

This path will also be displayed on the Files tab of the Containers page. It will be used when containers are executed.

4.Click Save.
