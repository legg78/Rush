Interchange upload data process

Process ID and name

10000913 - Interchange upload data process

Process function

The process uploads data on BINs, currencies or transactions to the folder dedicated for outgoing files of the related payment system.

Process parameters

Parameter

Description

Mandatory/Optional

Module

The name of the payment network.

M

Interchange load type

The type of data uploaded by the process:

•BINs

•Currency rates

•Operations

Note: For China UnionPay, the Interchange upload data process is configured only for BINs.

M

Interchange upload event

The event type related to the specified payment network.

M

Process timeout

The interval in seconds between the process start and completion.

M

Files

The process uses outgoing files of the text format.
