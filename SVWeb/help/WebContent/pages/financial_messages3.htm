Visa financial messages

The Financial Messages window displays financial messages generated when processing Visa authorizations.

1. Select Payment System > VISA > Financial messages.

Specify the search parameters in the search pane:

Parameter

Description

Session ID

Identifier of the file load session.

File name

Name of a message file.

ARN

Acquirer reference number in the authorization.

Date from and Date to

Initial and final dates of the period for which messages are displayed (selected in the calendar).

Card number

Number of the card (PAN) used for the authorization.

Terminal number

Number of the terminal where the authorization was executed.

Authorization code

Code provided by the issuer for the transaction response (approval or decline). The code is returned in the authorization response and is usually printed on the transaction receipt.

3. Click Search.

The search result displays the transactions matching the specified conditions. The following data is available for each transaction:

Indicates type of unattended terminal. The field must contain a space, or 1

Indicates the type of the prepaid card:

•Space = Not applicable

•P = Prepaid card

•L = Prepaid load

Service development

A numeric code indicating the type of commerce:

•1 = Electronic Commerce

•2 = In-Flight Commerce

•5 = Transponder-Initiated

•6 = Relationship Participant

•7 = Deferred Billing Indicator

Contains the response to an Address Verification Service (AVS) request, indicating matches or discrepancies between addresses and ZIP codes. The entry must contain a valid value. Refer to BASE II Clearing Data Codes for a listing of AVS Response Codes

Authorization source code

Indicates whether or not card present at authorization and type of commerce or service requested. The allowable entries are a space, A through Z, or 0 through 9.Refer to BASE II Clearing Data Codes for the list of codes that are defined for use by the members
