Configuring a rule set for processing

To configure a rule set for processing:

1.Create a rule set entity, and specify its category and procedures.

2.Create a bunch type and include the required transaction templates in it: specify the cash flows, settlements (for posting).

3.Assign the Post macros procedure to the rule set and specify the macro type.

4.Assign the rule set for event/operation processing.

oConfigure templates for operation processing: specify the rule set for the operation of the very type processing.

oSet parameters values for all procedures.

oIf the rule set is assigned for an event processing then add the rule set for the corresponding event.

oSet scales and modifies if required.

Note: For credit module processing bunch types should be assigned to the credit module events.
