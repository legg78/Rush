Viewing invoices

1.Click the Issuing > Credit > Invoices menu items.

 

2.Specify the search parameters:

Parameter

Description

Account number

Specify the account number for which you wish to view invoices.

Date from and Date to

Specify the date range for the invoices.

 

3.Click Search.

The search result will display the list of accounts and invoices in accordance with the specified parameters with information about invoices in them.

The detail panel displays all information about the selected invoice.

Tab

Description

Details

General information about the selected account and its invoice

Debts

Information about the account debt on the time of the invoice (date and time of the debt, source operation, debt amount, etc.)

Payments

Information about payments (date and time, payment amount, etc.)

Agings

Information about overdue payments
