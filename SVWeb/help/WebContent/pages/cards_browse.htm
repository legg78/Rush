Viewing cards

1.Click Issuing > Cards.

2.Specify the search parameters on the search pane.

Parameter

Description

Institution

Institution (card owner)

Name

The cardholder’s first name. If this field is specified, the Surname field must also be specified. Used to search for card belonging to one customer.

Surname

The cardholder’s surname. Used to search for cards belonging to one customer. Cards belonging to the customer whose name and surname match the entered sequence are displayed.

Customer number

The customer identifier. You can enter the identifier manually or select it from the Customer window by clicking the search icon ().

Card number

The card's primary account number (PAN). You can use a wildcard character (*) in combination with part of number. A list of cards that contain the specified sequence of numbers is displayed.

Card UID

The card unique identifier. This 11-digit numeric value is generated automatically when a card is issued if the system is configured accordingly. It serves to identify the card in external systems to comply with the security standards.

On the Issuing > Cards page, the unique identification number of the selected card is also displayed in Card UID column the list of cards and on the Details and Instances tabs. Besides that, card UID is available on all web pages displaying card data.

To search a card by its UID, you can use a wildcard (*) and type in a part of number.

Cardholder name

The cardholder’s name as embossed on the card. You can use a wildcard (*) and type in several characters. The cards belonging to customers whose names match the entered sequence are displayed.

Card type

Card product type as defined by the network to which the card belongs

Contract number

The number of the customer contract under which the card was issued (see the Contracts tab on the Customers > Customers page).

3.Click Search.

The search result pane displays a list of cards that satisfy the conditions. Controls to perform maintenance on the selected record appears right below the list.

•View number — view the full card number.

•Applications — switch to the Application page to view all registered applications for the selected card.

•Operations — perform voice operations from the list (for call center employees): change the card status, reset the number of entered incorrect PIN code, change the limit value, manual fee charging)

The details pane tabs display information about the selected card.

Tab

Description

Details

Information related to the selected card (the same data as the list of cards). The Check word button is used to check the selected card code word.

Addresses

Customer’s contact addresses

Additional

Additional cards parameters. These parameters are flexible fields that are used when generating reports.

Instances

List of the card instances, for example, a reissued card with the same number

Accounts

List of the card’s linked accounts and information about them (number, balance, currency, contract status and number, etc.)

Operations

List of the card operations and details about them (ID and type of operation, date, sum, status reason, etc.)

Service terms

List of all service terms and their values for the selected card in accordance with its contract.

You can change or set values for terms at the Product and object or Object definition level.

Cardholder

Personal data about the cardholder such as address, phone number, email

Customer

General and personal data of the customer (account owner)

Limit counters

Displays the current value of all limit counters, in accordance with the contract (service terms).

Cycle counters

Displays the current value of all cycle counters, in accordance with the contract (service terms).

Status logs

Information about the card status changes with information about the reasons for the changes.

Schemes

List of authorization schemes for the selected card

Documents

List of documents that are related to the selected card, for example scanned copies, reports, etc.

Applications

List of applications that are related to the selected card

Suites

List of fraud suites performed in operations with the card. The suites are attached as described in the similar procedure for an institution.

Reports

List of reports that are related to the selected card

Info

Additional information about the record selected in the main table
