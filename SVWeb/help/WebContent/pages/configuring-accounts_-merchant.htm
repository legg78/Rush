Configuring accounts, merchants, and terminals

This task explains how to assign services to acquiring objects such as accounts, merchants and terminals.

1. Specify services to be assigned to the objects in the respective tabs. The tabs Accounts, Merchants, and Terminals are displayed at the bottom of the main application parameter form.

2. Click Create.

If a service was not assigned to an entity, a No records found message is displayed in the corresponding window.

3. Specify the customer-specific services and click Next.

4. Specify the account-specific services and select the objects (Merchants, Terminals) that will have the selected service. Click Next.

If the application requires two or more accounts, services must be specified for each individual account.

5. Specify the merchant-specific services.

a.In the Parent Merchant parent block, specify the location of the current merchant in the hierarchy.

b.In the Accounts block, select the required account and click Next.

If the application requires two or more merchants, services need to be specified for each individual merchant.

6. Specify the terminal-specific services. Click Finish.

Note: If the application requires two or more terminals, services must be specified for each individual terminal.

 

To be able to modify the settings in the application wizard on the next step (Configuring parameters in the application wizard), select the Edit check box on the service allocation pages.
