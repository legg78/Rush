Adding account scheme templates

1.Click Acquiring > Configuration > Account schemes.

2.Specify search parameters in the search pane and click Search.

3.In the search result pane select the scheme to which you want to add the template.

4.Click the Account patterns tab and click Add.

5.On the New account pattern window, specify the parameter values for the account scheme template.

Parameter

Description

Operation type

Type of transaction for which the template is to be applied.

Reason

Transaction reason code.

Settlement type

Settlement type for which account scheme to be implemented.

Terminal type

Type of the terminal for which account scheme to be used.

Currency

Currency of the transaction.

Operating sign

The value corresponding to the direction of funds movement:

•Positive — funds are credited to the account

•Negative — funds are debited from the account

Merchant type

Category of the merchant for which the account scheme template is applied. The value is selected from the merchant type (MRCT) dictionary:

•Merchant for banking terminal registration

•Current merchant

•Terminal

Account type

Account type applied to the transaction.

Account currency

Currency of the account.

Priority

Priority applied to the selection of the current template.

6.Click Save.
