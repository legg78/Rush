Configuring main application parameters

To set the main parameters of an application.

1.Click Acquiring > Applications.

2.Click Create in the result search pane.

3.In the Create application window, configure values for application parameters:

Parameter name

Description

Institution

Institution identifier for which application is created.

Agent

Agent identifier associated with an application.

New customer

To create a new customer, select the New customer check box and set the customer type in the Customer type parameter.

Customer number

For an existing customer, select the customer his number using the icon.

Set the required values in the search filters and click Search. Leave the search filters empty to display all customers. Click the target customer record and click Select.

Customer type

This parameter is set only for new customers when the parameter New Customer is set.

Select the customer type from the list:

•ENTTAGNT - Agent

•ENTTCOMP - Company

•ENTTPERS - Person

New contract

To create a new contract, select the New contract check box and set the customer type in the Customer type parameter.

Contract

For an existing contract, select the contract number using the icon .

Set the required values in the search filters and click Search. Click the target contract record and click Select.

Contract type

Specifies the type of contract that will be created. This parameter is only set for new customers when the parameter New Contract is set.

Product

Select the Acquirer Product to use in the application.

Flow

Select the application creation flow from the list.

4. Click Create.
