Fee criteria

The Fee criteria page enables criteria for interchange fee application to be specified.

Parameters

Parameter

Description

ID

The fee identifier. The ID is automatically assigned by the system.

Modifier

The Boolean expression containing the application parameters of the fee. The expression is configured using the modifier wizard.

Fee

The fee created in the Fee settings. In the table, this is the link to the related fee record on the Fee settings page.

Name

The fee criterion name.

Operation Type

The type of the transaction for which the fee is calculated (according to the system dictionary of transaction types ─ OPTP).

Issuer Country

The card issuer country as defined by the relevant payment system.

Issuer Region

The card issuer region as defined by the relevant payment system.

Acquirer Country

A three-digit acquirer country code.

Acquirer Region

An acquirer region as defined by the relevant payment system.

Priority

A fee application priority ranging from 0 to 100, where 0 is the highest priority.

Start Date

The date when the fee becomes effective.

End Date

The date when the fee expires.

Fee type

The fee type defined according to the region:

•IAF — International Acquiring Fee

•ISA — International Standard Assessment

•OIF — Optional Issuer Fee

This parameter is available only for Visa fees.

Controls

Name

Function

Add

Creates a fee criterion.

Edit

Modifies the selected fee criterion parameters.

Delete

Removes the selected fee criterion.
