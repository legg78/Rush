Example: editing service terms values via an application

Task statement

The Sub product issuer product was configured (see the figure below).

 

There is a customer contract on servicing in accordance with the product (see the figure below): the account was opened.

It is necessary to change the Card issuance fee value (via application).

 

Task solution

1.Add an application: specify the customer and his contract number. Select the customer account to edit the service term value.

2.On the next window, select the check box of the service, which includes the term to edit.

3.Add an application element on the next page (service term for the service).

4.Specify values for the service term parameters.

5.Save the application. Set its status to Awaiting processing.

6.Process the application.

As a result, the service term value and validity time will be added (see the figure below) for the corresponding entity.
