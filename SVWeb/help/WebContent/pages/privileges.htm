Privileges

Privileges determine individual permissions that enable users to perform defined operations within the system. See the topic on Users, Privileges, and Roles for more information.

Note: The set of privileges for employees working with dispute applications is determined by the institution's requirements with respect to the dispute application life cycle.

1.Click Administration > Permissions > Roles.

2.Select the required user role.

3.On the Privileges tab, click Add and select role privileges.

4.Click Save.

Repeat the above steps for all privileges that need to be assigned to the users responsible for dispute application registration and processing. Privileges preconfigured in the system for dispute applications are listed below.

Privilege

Actions

Pages and controls

BRANCH_TEAM_USER

•View registered applications

•Assign the Ready for review status to an application

The user can send a selected application for review using on the Submit for review button.

CHARGEBACK_TEAM_USER

•Open a dispute application for review

•Accept an application

•Reject an application

The user can view applications submitted for review on the Operational rules > Disputes > Applications page, open an application,  accept or reject it using the following buttons:

•View

•Accept

•Reject

These buttons are available on the Applications page and in the opened application form.

VIEW_DISPUTE_APPLICATIONS

View registered applications

The user can view a list of dispute applications registered in the system on the Operational rules > Disputes > Applications page, view a selected application and submit it for review using the following buttons:

•View

•Submit for review

ADD_DISPUTE_APPLICATIONS

Add dispute application

The user can create a dispute application using the Add button on the Operational rules > Disputes > Applications page

MODIFY_DISPUTE_APPLICATIONS

Modify dispute application

The user can modify a dispute application using the Edit button on the Operational rules > Disputes > Applications page

REMOVE_DISPUTE_APPLICATIONS

The can remove a dispute application using the Delete button on the Operational rules > Disputes > Applications page
