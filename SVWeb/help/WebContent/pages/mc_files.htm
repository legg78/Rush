MasterCard Files

The Files window displays information regarding files of MasterCard financial messages.

1. Select Payment System > MasterCard > Files.

2. Specify the search parameters on the search pane.

Parameter

Description

Session ID

Authorization session number

File name

Name of a message file

Date from and Date to

Initial and final dates of the period for which messages are displayed.

To display all available messages, leave the filter fields empty.

3. Click Search.

The search result pane displays a list of files meeting the specified criteria. The following data is available for each file:

Parameter

Description

ID

File identifier.

File name

File name.

Session ID

Number of the session in which the file was loaded.

File date

File load date.

Institution

Institution name in the file.

Institution ID

Institution number in the file.

Network ID

Network number in the file.

Local file

Indicates that the file contains domestic transactions.

Additional information about a selected file is available in the tabs under the list:

Tab

Description

Details

The Details tab displays the same information as the list of files and additionally the following information:

Incoming

Indicates that the file is incoming.

Proc date

File processing date.

Session file id

File processing session number.

Rejected

Indicates that the file has been rejected.

rejectid

Reject reason code.

In addition, the tab contains information such as file reversal indicator, file ID, transmission ID, processing mode, file amount (checksum), file message counts, file header codes, and file trailer codes.

Financial messages

The Financial messages tab displays information on financial messages in the selected file (see list of messages).

Reject

The Reject tab contains rejection details (for rejected authorizations) such as message type identifier, reject codes and the name of the file containing the reject message).
