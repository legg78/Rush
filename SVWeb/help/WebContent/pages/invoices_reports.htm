Viewing available reports

1.Click Configuration > Reporting > Reports.

2.Specify search parameters and click Search.

The details pane tabs display additional data about the selected report.

Tab

Description

Details

General data of the report that was set on its creation.

Parameters

The parameters required to generate the report.

Output parameters

The output parameters.

Templates

The report templates that define the report data location.

Roles

The roles that have access to the report. Users assigned these roles are able to run the report.

Tags

The category of the report, for example, notification reports, MasterCard reports, etc.

Entities

The report entities. The tab displays data on the entity and the object types of the selected report.
