Setting ATM-specific parameters

When adding an ATM there are certain additional ATM-specific parameters that must be configured on the ATM tab. To configure these parameters:

1.Click Administration > Communication > Terminal templates.

2.Specify search parameters and click Search.

3.Click the required terminal template.

4.Click the ATM details pane tab and click Add.

5.Specify the ATM parameters.

Parameter

Description

ATM type

ATM terminal type.

Number of cassettes

Number of cassettes in the terminal.

Key change algorithm

Maximum number of bank notes that the reject cassette can hold (to avoid overfilling).

Reject dispenser warning

Minimum number of bank notes of any value. When the number of bank notes of any value reaches the specified value, the ATM sends a notification message to the host.

Receipt warning

Specifies when a warning must be sent to indicate that the receipt roll is about to run out. The value specified is the number of remaining receipts that can be printed.

Card captured warning

The maximum number of cards that can be captured before the terminal sends a corresponding message to its host to avoid overfilling.

Maximum number of notes

The maximum number of bank notes that can be dispensed in a single transaction.

Scenario

ATM scenario. The applicable scenario defines the transaction flows and screens that the ATM uses.

Number of coin dispensers

Number of coin dispensers in the terminal.

Periodic synchronization

Specifies whether synchronization is periodic. In the case of periodic synchronization, select the type of the transaction to count from the Count all operations for periodic synchronization list, and specify the period of synchronization (for example, after each 10 transactions) from the Count of operations for periodic synchronization list.

6.Click Save.
