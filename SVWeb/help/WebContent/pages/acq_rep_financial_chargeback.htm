Report on payment system chargeback transactions

Report name — Financial chargeback transactions with regard to payment systems.

Report description — The report is generated for chargeback transactions which are sent to the bank by the payment system in an incoming file. It is is also generated for presentment transactions sent by the Bank to the payment system in outgoing file. The Report is generated with regard to payment systems (VISA, Master Card, Union Card) as of the payment system clearing date (CPD) and is used for chargeback transaction accounting.

The report is generated after receiving a payment notification from payment systems.

Report header — Financial Chargeback Operations In payment system name By The Day Results For Clearing Period start date - end date.

Note: The report includes the transactions whose processing dates are within the specified period.
