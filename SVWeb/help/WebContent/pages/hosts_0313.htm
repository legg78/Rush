Hosts

Institutions are grouped into networks to ensure that they have the ability to interact with one another; for example, through message routing. The rules for operation, processing, and settlement are set within the network to which the institution belongs.

Institutions that form part of a network are registered as network members. Their settlements conform to the unified rules and standards mandated by the network. A single institution can be a member of several networks.

The default network can be set for a network member institution to unify  processing. This ensures that the network can be determined when the institution is only contained in inbound information. The network institution can thus be created without having a network assigned, allowing the partner bank or aggregator registration to be separated from the network assignment.

Note: A default institution also needs to be set for each network.

 

A member institution can be configured as a host for this network. The host enables a connection to the local network or international payment system and establishes the direct interaction among two institutions which build a settlement relationship for the transactions made between them.

The host owner is an institution providing the ability to interact with other institutions or networks. Also, a given institution can be the owner of several hosts within a network based on the host purpose (for example, one host for the issuing, another for the acquiring).

Communication standards such as the Authorization Interface (online) and the Clearing Interface (offline) are defined within the host. Thus, inter-host enables a connection to the local network or international payment system and establishes the direct interaction among two institutions which build a settlement relationship for the transactions made between them.

A host owner and a client institution must be defined during host configuration. The host (together with online and offline interaction standards) is assigned to the host owner. Standard online and offline parameters are also assigned to the client institution.

A network may contain none or any number of hosts.

Communication protocols may vary within the network depending on the device that sent the message. The Visa network, for instance, uses the VISA BASE I protocol for authorizations in general, and the VISA SMS protocol for ATM authorizations. Each protocol requires a corresponding routing configuration.
