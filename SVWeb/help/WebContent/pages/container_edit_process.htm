Editing process parameters in container

To change the process parameters in the container:

1.Click Administration > Processes > Containers.

2.Specify search parameters on the search pane and click Search.

3.Select the required container.

4.Go to the Processes tab on the details pane.

5.Select the process you need to edit and click Edit.

6.In the Edit container process window, make the required changes.

7.Click Save.
