Creating a dispute application

1.Click Operational rules > Disputes > Applications.

2.Click Add.

3.In the Create new dispute window, specify the application parameters:

Parameter

Description

Branch ID

Unique identifier of a branch to which the claim was submitted. This identifier is assigned according to the bank internal classification.

Card number

Number of the card used for the disputed transaction

Transaction date

Disputed transaction date

Currency

Disputed transaction currency

Amount

Disputed transaction amount

4.In the Files section, click Add.

5. On the Attach document page, specify the document type, browse for the file, and click Add.

6.Repeat step 5 for all documents that must be attached to the application. When all documents are added, click Close.

Note: A maximum of ten files can be attached to an application; no file must exceed 5 MB in size.

7.In the Create new dispute window, click Save.
