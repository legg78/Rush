Viewing registered terminals

1.Click Acquiring > Terminals.

2.Specify search parameters and click Search.

The search result pane displays a list of the registered terminals. The controls below the search result pane enable you to perform actions on the selected records.

Button

Description

Operations

Links to the Operations page to display a list of transactions from the terminal.

Applications

Links to the Application page to display a list of the acquiring applications of the terminal.

Reload

Updates the list of terminals and their statuses and parameters. Data is obtained from the database and new settings made on this page are sent to terminals.

Refresh

Updates the list of terminals, statuses, and parameters from the database.

Operation

Displays the transaction wizard used to perform manual transactions.

The details pane displays additional information about the terminal selected in the list. Information is organized into tabs.

Tab

Description

Details

General information about the terminal:

•Terminal ID — terminal’s external ID that is unique within the institution

•Terminal type — type of the terminal

•Merchant — link to the merchant that owns the terminal

•Instance number — card number (for imprinter only)

•Contract — contract defined with SmartVista that serves the terminal

•Installment support indicator — determines whether installment payment processing is supported at the POS terminal:

oYes — installment processing is supported

oNo — installment processing is not supported

•Other parameters from the terminal template

Additional

Additional parameters of the terminal. These parameters are flexible fields that may be used for reporting.

Connectivity

Communication device parameters related to the terminal. To set the parameters, click Configure and make necessary changes.

Keys

The terminal’s cryptographic keys. Use the shell (encryptionKeyChange command) to load keys onto the terminal.

Accounts

Information about the terminal accounts that were created using the contract. If required, you can create an adjustment.

Addresses

Addresses of the terminal.

Contacts

Contacts of the terminal and merchant.

Customer

Customer for whom the merchant (terminal owner) is registered.

Service terms

Service terms applicable to the terminal. You can change values of the terms that have either the “Product and object” or “Object” definition level only.

Notes

Notes about the terminal.

Limit counters

Information about current values of the counters that are used for limits.

Cycle counters

Information about current values of the counters that are used for cycles.

Standard versions

Communication standards used for the interaction between the terminal and the SmartVista system.

Payments

Additional settings used for payments.

Schemes

Authorization schemes applicable to the terminal.

ATM

Additional settings used for the ATM terminal type.

Dispensers

The list and parameters of the bank notes cassettes. This is specific to the ATM terminal type.

ATM CashIn

Parameters pertaining to deposited notes. This is specific to ATMs with cash-in function.

Status logs

History of the terminal status changes.

Revenue sharing

Parameters of fees which are applied on the terminal.

Voucher batches

Sets of slips produced by a terminal

Collection

Information about replenishments, with information about the terminal cassettes. This tab is specific to ATMs.

MCC redefinition

Rule used to change the terminal MCC.

Reports

List of reports related to the terminal.

Info

Additional information about the record selected in the main table.
