Creating arrays

An array is configured to store elements of a specific type.

1.Click Configuration > Arrays > Arrays.

2.On the Arrays page, click Add in the search result pane.

3.Specify the array parameters:

Parameter

Description

Language

Language of the input data.

Name

Name identifying the array.

Array type

The array type defines its applications.

Institution

Institution whose entities will use the array.

Agent

Institution agent.

Modifier

A modifier is a logical condition that compares a parameter to a concrete example. If the conditions for the selected modifier are met, the associated array that has been created will be used.

Is private

Determines whether this array can be accessed only by the user who created it or by all users.

Description

Additional information about the array.

4.Click Save.

5.Add the array elements.
