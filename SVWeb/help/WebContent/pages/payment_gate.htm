Payment gateway and payment settlements

A payment gateway is a host through which the system sends information about payments. A payment gateway can be one of the following:

•Payment provider (for service payments)

•Core banking system (for transfers within the bank)

•Processing center (for interbank transfers)

•International payment system (for cards plastic payments)

The diagram shows the “money flow” in which the payment gateway can be used for account replenishment, and for payment services.

 

A payment gateway is registered as a financial institution in the system and is configured as a host.

An application of a particular type that corresponds with the product is opened. The fee charging conditions for money fund transfers through the payment gateway is set at the product level. The fee rate may depend on which service provider must be paid and for what service.

It is recommended that the service providers be grouped and the fee set for the group instead of each service provider. However, in the case of large providers, fees can be set individually.
