Data on all successful transactions of internet shops supported by ASSIST system

Report name — Successful transaction data in the internets-shop table.

Report description — The report is generated for a specified period and displays the successful transactions of internet shops.

Report header — The data on all successful transactions of Internets-shops served in system ASSIST for the period start date - end date.

Report building parameters

Parameter name

Description

Start Date

Period start date. By default, the start date is the current date.

End Date

Period end date.

Terminal mask

Terminal ID mask.

Institution

A financial institution identifier.

Description
