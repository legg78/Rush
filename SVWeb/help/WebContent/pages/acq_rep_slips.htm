Slip list entered by operator

Report name — Slip list entered by operator.

Description — Generated for slip entry date with regards to card programs, currencies and operator codes. Slips are generated for each merchant in the SVBO slip generation module. The report is used for slip generation control and Processing Administration .

Report header — Slip List Entered By Operator № operator code On Operations In Currency currency code For Period start date - end date.

Note: The report which includes the slips entered during a specified period is analyzed during report generation.
