Configuring exchange rates

1.Click Configuration > Rates > Rates.

2.Click Add on the search result pane.

Exchange rate parameters can configured in either simple or advanced modes. By default, the simple mode page opens first. To change the mode, click Advanced mode.

3. Specify the rate parameters.

Parameter

Description

Institution

Institution to which the rate applies.

Rate type

The rate type of the institution to configure. Selected from the list of preconfigured rate types (see Creating rate types). A table of registered templates for the specified rate type and institution is displayed at the bottom of the window.

Effective date

The date on which a rate becomes effective and can be applied.

If two rates are effective on a specific date, the system uses the rate which was set last and whose effective date is closer to the settlement date. If a rate need not be applied, its effective date can be changed respectively.

Expiration date

Date on which the rate expires.

Source currency

Currency from which you are converting.

Source scale

A multiplier that can be applied to the source currency. It defines the number of source currency units.

Destination currency

The resulting currency.

Destination scale

A multiplier that can be applied to the destination currency.

Rate

The ratio at which the source currency is converted to the destination currency. Note that the effective rate is also affected by the source and destination scales. For example, in converting from U.S. dollars (USD) to Japanese yen, the source scale might be set to 1, the destination scale to 100, and the rate to 1.19. The effective rate is then USD 1 equals JPY 119.

Inverted

Select this option if the rate is reversible. If selected, a single exchange rate is entered. When performing a reverse conversion (from the destination currency to the source currency), the reciprocal of the defined rate (1/rate) is used. For example, if you specify a dollar–euro rate of 0.88 (with source and destination scales set to 1) and select the Inverted check box, the effective rate for reverse conversion is 1.1363.

IMPORTANT: Exchange rates must be explicitly set for non-reversible rate types (such as customer rates).

3.Click Save.

If the rate was included into the rate pair, additional options are displayed in the simple mode page after specifying the Institution and Rate type.

 

Parameter

Description

Label

Rate pair identification option. If this check box is selected, a label for the rate pair is displayed on the Rates page.

Source

Source currency identification.

Destination

Destination currency identification.

Rate example

Example of the rate format.

New rate

Newly defined rate.
