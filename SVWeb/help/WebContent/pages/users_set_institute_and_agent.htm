Assigning institutions and agents to users

Users can be assigned to institutions manually or by using an application (see User management using an application).

To assign a user manually:

1.Click Administration > Permissions > Users.

2.Click Search on the search pane.

3.Select the required user from the search result pane.

4.Click the Institutions tab on the details pane.

5.To assign all agents of the institutions to the user, select the institutions, and click Set default.

The user gains access to the child institutions and their agents too. If a new child institute is displayed, access is granted automatically.

6.To make changes within the institution, click Institutions.

Parameter

Description

Access granted

Specifies that the user has access to the specified institutions. It is recommended that the institution’s agents are explicitly specified; there are no primary agents selected by default.

Entirely

Specifies that the user has access to all the agents of the selected institution. Individual agents cannot be specified if this check box is selected.

The option is active if the Access granted check box for the current institution is selected. To deny access to a specific agent, clear the check box and specify the required agents.

Default

This check box is informational. Its value is set on the Institutions tab.

7.Select the check box that corresponds to the relevant institution to grant access to it, and click Close.

8.To assign agents to the user selectively, select the institution, and click Agents.

9.Select the relevant check box.

Parameter

Description

Access granted

Agents to which the user will have access.

Default

Agent that is to be the default one for the user.

10.Click Save agents to store settings. The changes will take effect when the configuration is reloaded.
