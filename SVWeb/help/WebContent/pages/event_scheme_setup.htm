Creating event schemes

1.Click Configuration > Notification > Schemes.

2.Specify search parameters on the search pane and click Search.

3.Select the required notification scheme from the search result pane.

4.Click the Scheme events tab on the details pane.

5.To add a new event scheme, click Add.

6.Specify the scheme parameters.

Parameter

Description

Scheme

Scheme name used in the system.

Event type

Type of event that should be notified. The possible values are the CYTP and EVNT dictionary articles.

Entity type

Type of the entity for which the event occurred.

Contact type

Recipient contact type to which the notification will be sent.

Notification

Notification rules for the event type.

Channel

Method for delivering the notification; for example, by email or phone.

Scale

Scale that is used to configure special conditions applicable to the generation and submission of the notification.

Delivery period

Time period allowed for notification delivery.

Customizable

Specifies whether the notification is editable by clients/users.

Batch send

Reserved for future use.

Status

Status of the notification of the event type.

Priority

Priority applied to the notification scheme (1 is the highest priority).

7.Click Save.
