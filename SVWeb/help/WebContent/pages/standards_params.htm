Setting standard parameters

1.Click Administration > Communication > Standards.

2.Specify the search parameters and click Search on the search pane.

3.Select the standard / standard version to set parameter value.

4.Click the Parameters tab on the details pane and set following values:

oDefault value

i.Select the standard on the search result pane.

ii.Select the parameter on the details pane.

iii.Click Add.

iv.Set the default value.

 

oValue for standard version:

i.Select the standard version on the search result pane

ii.Select the parameter on the details pane

iii.Click Set.

iv.Set the value for standard version.

5.Click Save.

To delete the value of the protocol parameter, select the parameter and click Delete. Click OK in the alert dialog window to confirm deletion.

Note: Be sure that the ACQUIRER_BIN and BUSINESS_ICA parameters are configured correctly for the MasterCard standard. These values are used to determine the acquiring institution by the acquiring BIN in the financial message in accordance with the IPM clearing formats.
