Creating a naming format

1.Click Operational rules > Naming > Formats.

2.On the search result pane, click Add.

3.Specify the parameter values for the new naming format.

Parameter

Description

Language

Language to use for the format name.

Name

Name of the format.

Institution

Institution with which the naming format is associated.

Entity type

Entity type associated with the naming format.

Name length

Required length of the naming format.

Pad type

Padding type defining whether the format should be left or right padded using the specified string.

Pad string

String to use for padding.

Check algorithm

Algorithm used to verify the check digit.

Check position

Check digit position of the naming format.

Check base position

Start position to generate the check digit.

Check base length

Check digit base length (for instance, if the first 13 digits are used to generate the check digit then Check base position should be 1 and Check base length should be 13).

Index range

Select a range to use for components from the Index range list. To create a new range, click the Add link and specify the range parameters.

Use for checking

Select the Use for checking check box to verify whether the entity number (input manually, obtained from an external system, or generated automatically) complies with the naming format or not.

4.Click Save.
