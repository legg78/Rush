Viewing accounts

To view information related to a specific account:

1.Click Issuing > Accounts.

2.Specify the search parameters on the search pane.

Tab

Description

Institution

Institution under which the account was opened.

Status

Account status.

Account number

Account number.

Type

Account type.

Customer number

Customer identifier. Enter the value manually or select from the Customer window that appears when clicking the icon.

Currency

Account currency code.

Contract number

Customer contract number under which the account was opened.

3.Click Search.

The details pane tabs display information of the selected account.

Tab

Description

Details

General information about the selected account.

Additional

Additional account parameters. These parameters are flexible fields and are used when generating reports.

Balances

List of the account balances and current information for each of them.

Transactions

Processed transactions. The information is displayed by account balances.

•To display the desired balance transaction, select its check box in the left side of the pane and click Show entries.

•To display the entries for several balances, select the balances you want to view and then click Show entries.

Operations

List of the account transactions and information about them, including ID and transaction type, date, sum, status, and status reason.

Service terms

List of all service terms and their values for the selected account in accordance with its contract. The values can be changed or set at the Product and object or Object definition levels.

Cards

List of cards assigned to the account, with information about them including the account and card number masked, the cardholder name, product, and registration date.

Customer

Information about the customer (account owner).

Limit counters

Current value of all limit counters in accordance with the contract (service terms).

Cycle counters

Current value of all cycle counters in accordance with the contract (service terms).

Loyalty bonuses

Information about all loyalty bonuses pertaining to the account.

Notes

Additional comments about the selected account.

Documents

List of documents that are related to the selected account; for example, scanned copies and reports.

Status logs

Information about the account status changes with information about the reasons for the status changes.

Applications

List of applications related to the selected account.

Suites

List of fraud suites performed in operations with the account. The suites are attached as described in the similar procedure for an institution.

Reports

List of reports related to the selected account.

Info

Additional information about the record selected in the main table.

Credit details

Information about account credits (for the credit service).
