Example: Configuring a template for the limit setup transactions

This topic describes how to configure processing templates for the limit creation and change transactions:

1.Click Operational rules > Processing > Templates.

2.On the search result pane, select the template for the transactions of the OPTP1001- Providing credit limit type.

3.On the Rules details pane, add the following rules to the template:

oSelect object account. This rule is applied to determine the account which limit must be created or changed

oChange account balance. This rule is applied to update the account balance following the transaction processing

oSet fee object. This rule provides creating:

oa fee of the specified type for the determined account.

oa new fee range. The fee rate will be equal to the transaction amount

oCredit limit increase. This rule provides increasing the limit of the selected account.

4.Configure parameters of the rules in the Parameters section of the Rules details pane.

Note. This configuration is applicable for the template а transactions of the OPTP0403 -Set entity limit type.
