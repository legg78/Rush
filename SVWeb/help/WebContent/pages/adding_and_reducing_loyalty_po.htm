Adding and deducting loyalty points as an operator

Operators can add loyalty points to a specific account or deduct points from it using the Operation wizard in the Back Office user interface.

1.Click Issuing > Cards.

2.Select the card and click Operation.

3.On the Select operation type page, select Manual enrollment and spending of loyalty points from the Operation Type list, and click Next.

4.On the Data input page, select the card and specify values for the following parameters:

Parameter

Description

Operation type

The type of transaction for which points are added or deducted.

Operation reason

Provide a reason for adding or deducting loyalty points.

Operation currency

The transaction currency.

Operation amount

The transaction amount.

5.Click Next.

6.On the Result page of the operation wizard, click Finish.
