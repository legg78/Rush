Creating processes

1.Click Administration > Processes > Processes.

2.Click Add on the search result pane.

3.Specify the process parameters.

Parameter

Description

Language

Language of the input text data.

Name

Process name.

Description

Any description of the process set on its registration.

Procedure name

Name of the procedure that pages included in the process.

Institution

Institution for which the process can be performed.

Parallel

Process can be performed in several flows.

External

Process can be run from an external system (via Java means).

Interrupt threads

Determines whether all process threads are interrupted or not when at least one of them fails.

If this check box is selected, the system tests for an interrupt and exits the thread if an error is received: the process status will be PRSR0003 - Failed. In case of failure, the cause of the interruption can be determined.

If the check box is not selected and an interrupt occurs, the process status will be PRSR0004 - Finished with errors.

4.Click Save.

5.Specify the process parameters and files.
