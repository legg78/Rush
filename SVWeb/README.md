# Quick Start

### Setup Git

[Install git client and configure access to repository](http://gitlab.bt.bpc.in/DevOps/sv_installer/wikis/install-git)

### Build project

Weblogic application:

    $ cd  svweb/
    $ mvn clean package

Websphere application:

    $ cd svweb/
    $ mvn clean package -P dev,websphere
