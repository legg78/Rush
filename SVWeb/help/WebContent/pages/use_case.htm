Example: Configuring an Interchange fee rule

Requirement

Configure an interchange fee rule for MasterCard POS transactions.

Conditions

•Both issuer and acquirer are from different countries within the European Economic Area.

•The card product is a MasterCard standard consumer credit card.

•The fee is a fixed value of 5 euro cents plus 0.14% of the transaction amount.

Steps

1. Create a fee (see Creating a fee). For the Type, select Percent + fixed value.

2. Specify the fee criteria (see Specifying fee criteria) according to the requirements of the Interchange Manual Europe Region 13 May 2014 document where:

•Chapter 4 “Intercountry Fallback Fee Rates” defines the requirements for the issuer and acquirer location.

•Paragraphs “Intra-EEA MasterCard POS Fallback Interchange Fee Rates” and “Intra-EEA MasterCard Purchase with Cash Back Fallback Interchange Fee Rates” define the requirements for the transaction type.
