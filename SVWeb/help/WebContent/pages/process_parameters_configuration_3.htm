Setting process parameters

To set parameters for the process:

1.Click Administration > Processes > Processes.

2.Specify search parameters on the search pane and click Search.

3.Select the process on the search result pane.

4.Click the Parameters details pane tab.

5.Make the required action:

•To add a parameter to the list, click Add on the tab.

•To edit the parameter in the process, select it and click Edit.

•To delete a parameter, select it and click Delete.

When you add or edit a parameter, the following attributes should be specified:

Attribute

Description

Parameter

Parameter system name.

To set a value:

1.Click Add.

2.Click Search.

3.Select the parameter and click Select.

Name

Parameter name. Automatically set after the parameter selection.

Display order

Parameter order in the display list.

Default value

Parameter default value. It is used on processing if was not specified on the process running.

Need format

Parameter format is required before the process is run.

•This attribute is always used for data of the date and number types.

•If the data type is symbol and the Need Format check box is selected, the system uses the Default value as the parameter value.

•If the data type is symbol and the Need Format check box is not selected, the system uses the Default value as a function for the parameter value calculation.

Mandatory

This parameter value must be specified for successful processing.
