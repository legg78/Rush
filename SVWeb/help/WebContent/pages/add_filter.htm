Adding search filters

Filters can be used to quickly access data. Filters contain search parameters defined by the user.

Note: A filter is attached to the page and can be displayed on the Bookmarks tab (depending on the user setting).

A user can create a new filter, or load an existing filter.

To create a new filter:

1.Specify search parameters and click Filter.

2.Select Save filter.

3.In the Save filter window, specify the filter parameters:

Parameter

Description

Type

Select the new option to save the filter as a new one, or the Use existing filter option to modify a previously loaded filter.

Name

Specify the filter name (available when adding a new filter).

Display order

Setting the display order number determines the order in which the new filter will be displayed relative to any existing filters.

4.Click Save.

Note: Filters are stored in the database per page per user.

 

To load an existing filter:

1.Click Filter on the page search pane:
 

 

2.Select Load filter.

3.Select the filter from the list.

4.Click Set.

If the Search automatically check box was selected, the system will search the database for the selected filter parameters. Otherwise, it will only fill the search parameters values of the selected filter in the search pane.
