List of cash disbursement operations

Report name — List of cash disbursement operations.

Report description — DMS operations List of cash disbursement operations.

A report is generated on all payment systems (Union, Master, Visa, Diners) for the SVBO clearing date. It contains cash acquiring DMS operations for a specified period. The report shows only processed operations by currencies, agents, and merchants.

It is used for settlements with agent banks on CASH/SALE acquiring operations, comparison with performed operations in ATMs, and for report checking. The report is intended for the following organizations:

•Processing administration

•HO bank transaction administration

•Branch

•Additional bank offices

•Agent banks

•Daughter banks

Report header — DMS operations agent name. The register on operations of delivery of cash on currency currency code, processed from start date to end start date, on merchant merchant name, merchant code merchant ID, The address: merchant address, phone: merchant phone
