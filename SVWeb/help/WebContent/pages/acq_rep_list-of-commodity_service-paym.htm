List of commodity/service payment operations

Report name — Financial list sale.

Report description — DMS operations List of trade operations.

A report is generated on all payment systems (Union, Master, Visa, Diners) for an SVBO clearing date.The report is only generated for processed DMS operations with regards to currencies and agents.

It is used for settlements with merchant on SALE-acquiring transactions; settlements with agent banks on CASH/SALE-acquiring transactions; and comparison with transactions performed on ATM devices and account verification.

The report is intended for the following organizations:

•Processing administration

•HO bank transaction administration

•Branches

•Additional offices

•Agent banks

•Daughter banks

Report header — DMS operations agent name. The register of operations on payment for goods/services on currency code, processed from start date to end start date, on merchant name, merchant ID, The address: merchant address, phone: merchant phone
