Viewing applications

1.Click Issuing > Applications.

2.Specify the search parameters on the search pane.

Parameter

Description

Institution

Institution for which the target application was registered.

ID

Application identifier.

Status

Current status of the application.

Card number

Card number (PAN) of the application.

Customer number

Customer ID of the application.

Account number

Account number of the application.

Date from

Date on which the application was registered.

Application number

Unique number identifying the application.

3.Click Search.

The search result pane displays the following information:

Parameter

Description

ID

Application identifier.

Flow

Application status defining a set of its processing stages and possible transitions between them.

Description

Additional information about the application.

Status

Current status of the application.

Created

The date when the application was registered.

Last updated

Date on which the application was last updated.

Application number

Unique number identifying the application.

Institution

Institution for which the target application was registered.

 

The controls below the list are used to manage applications.

Button

Description

Create

Create a new application via the SmartVista user interface.

Edit

Edit data of the selected application. The button is only available if the application status is Just created or Awaiting processing.

View

View the application. The button is only available if the application status is Processed successfully.

Delete

Delete the application. The button is only available if the application status is Processed successfully.

Process

Process the application. The button is only available if the application status is Awaiting processing.

The set of tabs display additional information about the selected application:

Tabs

Description

Details

Application parameters.

Errors

Any errors encountered during the last application processing.

Notes

Notes made by application.

History

Application processing log.

Trace

Information on the application-related events tracking in the system.

Linked objects

List of system entities (cards, accounts, customers) related to the application.
