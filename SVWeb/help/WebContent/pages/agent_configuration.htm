Managing agents

1.Click Structure > Bank organization > Agents.

2.Specify the search parameters:

Parameter

Description

Institution

Agent institution

Name

Agent name

Type

Agent type

External number

Agent number as used in the external system.

Parent

The agent that is the parent of the current agent.

3. Click Search. The search result pane displays the list of agents matching the search parameters:

Parameter

Description

Name

Agent name

Type

Agent type

ID

Unique agent identifier

External number

Agent number as used in the external system.

 

The following control buttons are available:

Button

Description

New

Creates a new agent.

Edit

Changes the selected agent parameters.

Delete

Deletes an agent from the system.

The details pane displays the following tabbed information for the selected agent. You can manage this information.

Tab

Description

Details

General agent parameters

Accounts

Agent accounts

Addresses

Agent addresses

Contacts

Agent contacts

Settings

Agent settings

Notes

Notes made by the system user.

Schemes

Authorization schemes applicable to the agent.

Association

Association with customer. Used to register the institution’s terminals.
