Flexible fields

Flexible fields are used to register additional parameters and set values for them. Flexible fields can be registered in the system in order to provide additional parameterization of system entities.

Unlike essential parameters, additional parameters can be configured according to the characteristics of the organization. These parameters do not take part in processing and are typically used for reporting and analysis. For example, you may need to add a bank identification code (BIC), or record the maiden name of a customer’s mother.
