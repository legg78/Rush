Configuring an acquirer product

1.Click Acquiring > Configuration > Products.

2.Click Add.

3.Specify general parameters for the new product.

Parameter

Description

Institution

The institution that uses the product.

Parent product

The parent product from which the new one inherits its settings. If the product has no parent, leave the field empty.

Contract type

The contract type for which the product is available.

Status

The status of the new product once it is created. If the status is Active then the product can be used immediately after creation.

Language

The language of the input data.

Name

The name of the product.

Product number

The number of the product used for external systems. If it is not specified, this number is set equal to the identifier.

Description

A description of the product.

4.Click Save.

5.Define services for the product.

6.Set values for the required service terms.

7.Specify the account types for the product.
